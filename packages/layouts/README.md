# Register Site Layouts

> Deze package bevat de generieke layouts voor een register site.

[![NPM Version](https://img.shields.io/npm/v/%40developer-overheid-nl%2Fdon-register-layouts)](https://www.npmjs.com/package/@developer-overheid-nl/don-register-layouts)
![NPM Last Update](https://img.shields.io/npm/last-update/%40developer-overheid-nl%2Fdon-register-layouts)

## Gebruik

Deze package wordt gebruikt in de register sites van developer.overheid.nl en in register-site-template repository.

De package heeft o.a. de volgende dependencies:

- [Rijkshuisstijl Community](https://www.rijkshuisstijl-community.nl/) voor de React componenten, CSS en design tokens
- [Utrecht Component Library](https://www.npmjs.com/package/%40utrecht%2Fcomponent-library-css)
- [@developer-overheid-nl/don-register-components](https://www.npmjs.com/package/@developer-overheid-nl/don-register-components)
- [@developer-overheid-nl/don-register-locales](https://www.npmjs.com/package/@developer-overheid-nl/don-register-locales)
