# Register Site Locales

> Deze package bevat het localization-bestand voor de (kleine) teksten van een register site.

![NPM License](https://img.shields.io/npm/l/%40developer-overheid-nl%2Fdon-register-locales)
[![NPM Version](https://img.shields.io/npm/v/%40developer-overheid-nl%2Fdon-register-locales)](https://www.npmjs.com/package/@developer-overheid-nl/don-register-locales)
![NPM Last Update](https://img.shields.io/npm/last-update/%40developer-overheid-nl%2Fdon-register-locales)

## Gebruik

Deze package wordt gebruikt in de register sites van developer.overheid.nl en in register-site-template repository.

De package heeft o.a. de volgende dependencies:

- [i18next](www.i18next.com) internationalization framework
