# developer.overheid.nl Register Sites Proprietary Example

Deze package bevat assets die binnen de developer.overheid.nl register sites gebruikt kunnen worden, die niet open source zijn. Fonts, logo's en iconen bijvoorbeeld. Hierdoor kan de rest van de codebase blijven werken en kan deze package vervangen worden door een eigen implementatie met eigen (proprietary) assets.

In andere packages is de package `@developer-overheid-nl/proprietary` als dependency opgenomen. **Deze package wordt in PNPM als alias naar deze package verwezen.** Hierdoor kan de rest van de codebase gewoon werken.
