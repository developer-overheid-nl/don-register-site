# Formulier om API-key aan te vragen toegevoegd

In app `api-register` is een formulier toegevoegd waarmee gebruikers een API-sleutel kunnen aanvragen. 

## Nieuwe componenten

- Button
- Fieldset
- FormFieldLabel
- FormFieldTextInput

## Nieuwe layouts

- BasicWithAside

## Wijzigingen

- ReadOnlyTextInput: fontVariant toegevoegd.
- i18n: nieuwe vertalingen toegevoegd voor foutmeldingen bij het aanvragen van een API-sleutel.
